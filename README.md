# auto_model

🚗 AutoLiquid — прогноз ликвидности автомобилей с пробегом

AutoLiquid — это веб-приложение на Python, которое не только оценивает рыночную цену автомобиля, но и предсказывает срок его продажи (ликвидность) и даёт рекомендации по цене.

📌 Описание проекта

В отличие от стандартных калькуляторов (основанных только на годе и пробеге), AutoLiquid использует машинное обучение и учитывает широкий набор факторов:

марка и модель автомобиля
год выпуска
пробег
состояние (1–10)
цвет
количество владельцев
наличие ДТП
сезонность (месяц продажи)

📊 Результат:

прогноз цены
диапазон цены (доверительный интервал)
срок продажи (в днях)
оценка ликвидности
